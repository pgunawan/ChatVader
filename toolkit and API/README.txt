HOW TO BUILD PATH AND TAGGERS FOLDER ON ECLIPSE

BUILD PATH
In order to run the program properly, you will need to configure build path and include the provided .jar files into Eclipse.

In Eclipse, when you open Eclipse, and already have all the required classes in a project, you will need to right click the project, click Build Path --> Configure Build Path. 
Open library tab, and click on "Add External JARs" on the right side of the window.

BUILD TAGGERS FOLDER
You need to have the taggers folder in order to run the POS tagger. 
Right click on the project, create a new folder and name it "taggers". Inside the file, you should have both .tagger and .props file.
